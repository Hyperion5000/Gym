        ❓ Часто задаваемые вопросы

        Что такое ТМ (тренировочный максимум)?

        ТМ — это вес, с которым вы можете выполнить примерно 5-8 повторов с хорошей техникой. Обычно это 85-90% от вашего максимума на 1 повтор. ТМ используется для расчета RIR (запаса повторов) и рекомендаций по весу.

        Важно: ТМ требуется только для основных упражнений типа A (присед, жим, тяга). Для аксессуарных упражнений (типы B, C, D) ТМ не требуется — просто вводите вес и повторы во время тренировки.

        Для упражнений типа A: ТМ автоматически рассчитывается как 90% от лучшего e1RM за последние 4 недели и устанавливается при первом сохранении сета. Вы можете вручную изменить ТМ, нажав на кнопку ⚙️ рядом с ТМ.

        Не знаете свой ТМ? При первом запуске приложение попросит вас ввести вес и количество повторов для упражнений типа A (например, 80кг × 6 повторов). На основе этих данных приложение автоматически рассчитает ваш e1RM и ТМ. Вам не нужно знать точный максимум на 1 повтор!

        Что такое RIR?

        RIR (Reps in Reserve) — запас повторов до отказа. Например, RIR 2 означает, что вы могли бы сделать еще 2 повтора.

        Для упражнений типа A: RIR автоматически рассчитывается на основе вашего ТМ, веса и повторов.

        Для аксессуарных упражнений (типы B, C, D): RIR не рассчитывается, так как ТМ не требуется.

        Цветовая индикация (только для типа A):

        • 🟢 Зеленый: попали в целевой RIR (±0.5)
        • 🟡 Желтый: близко к цели (±1.5)
        • 🔴 Красный: далеко от цели (>1.5)

        Что такое RPE?

        RPE (Rate of Perceived Exertion) — шкала воспринимаемого усилия от 1 до 10. Рассчитывается как: RPE = 10 - RIR

        Например: RIR 2 = RPE 8

        Что такое e1RM?

        e1RM (estimated 1 Rep Max) — расчетный максимум на 1 повтор. Вычисляется по формуле Epley: e1RM = вес × (1 + повторы / 30)

        Используется для отслеживания силового прогресса.

        Типы упражнений A/B/C/D — что это?

        • A — Основные силовые упражнения (присед, жим, тяга). Требуется ТМ для расчета рекомендаций по весу и RIR.
        • B — Тяжелые вспомогательные упражнения. ТМ не требуется — просто вводите вес и повторы.
        • C — Изоляция, высокоповторные упражнения. ТМ не требуется — просто вводите вес и повторы.
        • D — Вспомогательные упражнения для квадрицепсов. ТМ не требуется — просто вводите вес и повторы.

        Почему ТМ только для типа A? Для основных упражнений важно точно дозировать нагрузку на основе ТМ. Для аксессуарных упражнений достаточно подбирать вес, с которым вы можете выполнить заданное количество повторений с правильной техникой.

        Как работают недели 1-4?

        Программа построена на мезоциклах по 4 недели с прогрессивной перегрузкой:

        • Неделя 1: Высокий RIR (3-4) — адаптация
        • Неделя 2: Средний RIR (2-3) — нагрузка
        • Неделя 3: Низкий RIR (1-2) — пиковая нагрузка
        • Неделя 4: Высокий RIR (4-5) — восстановление (deload)

        Как работает автоматический расчет веса и повторов?

        Для упражнений типа A: При вводе веса или повторов приложение автоматически рассчитывает недостающее значение на основе ТМ и целевого RIR:

        • Если введены только повторы — автоматически рассчитывается вес
        • Если введен только вес — автоматически рассчитываются повторы

        Для этого должны быть заданы ТМ и целевой RIR для упражнения.

        Для аксессуарных упражнений (типы B, C, D): Просто вводите вес и повторы вручную. Автоматический расчет не применяется, так как ТМ не требуется.

        📝 Кнопка "Заполнить" (в подсказках): подставляет вес и повторы из последней тренировки этого упражнения.

        Как работает автосохранение?

        Данные сохраняются автоматически при заполнении сетов (с задержкой 2 секунды после последнего ввода). Вам не нужно нажимать кнопку "Сохранить" — все данные сохраняются в базу данных автоматически.

        Как добавить или удалить сет?

        Используйте кнопку "+ Добавить сет" под каждым упражнением для добавления нового сета. Кнопка удаления ❌ появляется рядом с каждым сетом (кроме первого, если сет только один).

        Как записать упражнение с собственным весом (подтягивания)?

        Для упражнений с собственным весом введите 0 в поле "Вес (кг)". Приложение корректно обработает это значение и рассчитает e1RM и RIR.

        Зачем нужна кнопка "Завершить тренировку"?

        Кнопка "✅ Завершить" появляется после сохранения хотя бы одного сета. При нажатии:

        • Тренировка помечается как завершенная
        • Сохраняется статистика (количество сетов, тоннаж)
        • Данные учитываются в дашборде

        Как сделать резервную копию данных?

        Нажмите кнопку 📥 Экспорт в верхнем меню. Скачается JSON файл с вашей базой данных.

        Рекомендация: делайте экспорт раз в неделю и сохраняйте файлы в облако (Google Drive, iCloud и т.д.)

        Для восстановления: нажмите 📤 Импорт и выберите файл.

        Данные хранятся локально или на сервере?

        Полностью локально! Все данные хранятся в вашем браузере (IndexedDB). Ничего не отправляется на сервер.

        Внимание: если очистите данные браузера — потеряете тренировки. Поэтому важно делать экспорт.

        Работает ли приложение офлайн?

        Да! После первой загрузки приложение работает полностью офлайн. Интернет нужен только для:

        • Первой загрузки SQL.js (~2 МБ)
        • Обновления приложения

        Как установить приложение на iPhone?

        1. Откройте сайт в Safari
        2. Нажмите кнопку "Поделиться" (квадрат со стрелкой вверх)
        3. Выберите "На экран «Домой»"
        4. Готово! Приложение появится на главном экране

        Что происходит при первом запуске приложения?

        При первом запуске приложение покажет экран онбординга.

        Важно: ТМ требуется только для упражнений типа A (присед, жим, тяга). Для аксессуарных упражнений (типы B, C, D) ТМ не требуется — просто вводите вес и повторы во время тренировки.

        Для упражнений типа A: Вам не нужно знать точный тренировочный максимум (ТМ)! Просто введите:

        • Вес — какой вес вы можете поднять (например, 80кг)
        • Повторы — сколько повторов вы можете сделать с этим весом (например, 6 повторов)

        Приложение автоматически рассчитает ваш e1RM и ТМ на основе введенных данных. Например, если вы введете 80кг × 6 повторов, приложение рассчитает e1RM ≈ 96кг и ТМ ≈ 86.4кг.

        Для аксессуарных упражнений (типы B, C, D): Поля для ввода веса и повторов не показываются в онбординге. Просто вводите вес и повторы во время тренировки — ТМ не требуется.

        Совет: Не обязательно заполнять все упражнения типа A сразу. Можно указать только те, которые вы знаете, и добавить остальные позже.
