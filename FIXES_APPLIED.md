# Исправления применены

## Критическая проблема решена

### Проблема
Приложение не загружалось на iPhone 11 с ошибкой:
```
Identifier 'roundToStandardPlates' has already been declared
```

### Решение
1. **Убрана локальная константа `roundToStandardPlates`**
   - Функция теперь объявлена только в `window.roundToStandardPlates`
   - Все вызовы обновлены для использования `window.roundToStandardPlates`

2. **Добавлена защита от повторной загрузки модуля**
   - Проверка `window.__MESO_MODULE_LOADED__` перед выполнением кода
   - Защита от повторного вызова `initApp()`

3. **Обновлен service worker**
   - Версия кеша обновлена до v8
   - Добавлена принудительная очистка старых кешей
   - Отключено кеширование service worker (`updateViaCache: 'none'`)

4. **Исправлена ошибка с `initExportReminder`**
   - Убран вызов несуществующей функции
   - Заменен на прямой вызов `checkExportReminder`

## Статус

✅ Ошибка `roundToStandardPlates` исправлена
✅ Функция `roundToStandardPlates` доступна в `window`
✅ Приложение инициализируется (`appInitialized: true`)
✅ Защита от повторной загрузки модуля работает

## Рекомендации

Если приложение все еще не загружается полностью:
1. Очистите кеш браузера (Ctrl+Shift+Delete)
2. Отключите service worker в DevTools → Application → Service Workers
3. Обновите страницу с принудительной перезагрузкой (Ctrl+F5)
