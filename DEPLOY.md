## 🌐 ДЕПЛОЙ НА GITHUB PAGES

### Шаг 1: Создание репозитория

1. Перейдите на [github.com](https://github.com)
2. Создайте новый репозиторий:
   - **Name**: `gym-tracker` или `meso-app` (любое имя)
   - **Visibility**: Public (или Private, если хотите)
   - ✅ Отметьте **"Add a README file"** (опционально)

### Шаг 2: Загрузка файлов

1. Загрузите все файлы проекта в репозиторий:
   - `index.html`
   - `app.js`
   - `db.js`
   - `style.css`
   - `schema.sql`
   - `plan.csv`
   - `service-worker.js`
   - `manifest.json`
   - `icon-192.png`
   - `icon-512.png`
   - `apple-touch-icon.png`
   - `README.md`
   - `.gitignore`

2. **НЕ загружайте:**
   - `REVIEW.md` (можно удалить или оставить)
   - `DEPLOY.md` (можно удалить или оставить)

### Шаг 3: Включение GitHub Pages

1. В репозитории откройте **Settings** → **Pages**
2. В разделе **"Source"**:
   - Выберите **Branch: main** (или `master`)
   - Папка: **/ (root)**
3. Нажмите **Save**

### Шаг 4: Ожидание деплоя

1. Подождите 1-2 минуты
2. Обновите страницу Settings → Pages
3. Должна появиться ссылка: `https://USERNAME.github.io/REPO-NAME/`

### Шаг 5: Проверка

1. Откройте ссылку в браузере
2. Должен появиться индикатор загрузки
3. После загрузки SQL.js должен появиться интерфейс
4. Проверьте работу:
   - Выберите неделю и день
   - Введите данные
   - Сохраните

## 📱 ИСПОЛЬЗОВАНИЕ НА IPHONE

### Шаг 1: Открытие сайта

1. Откройте **Safari** на iPhone
2. Перейдите на ваш GitHub Pages URL:
   ```
   https://USERNAME.github.io/REPO-NAME/
   ```
3. Дождитесь загрузки (первый раз может быть долго ~2 МБ SQL.js)

### Шаг 2: Установка PWA

1. После загрузки сайта нажмите кнопку **Поделиться** (квадрат со стрелкой вверх)
2. Прокрутите вниз и найдите **"На экран «Домой»"**
3. Нажмите - приложение установится

### Шаг 3: Первый запуск

1. Откройте приложение с главного экрана
2. Должен появиться индикатор загрузки
3. При первом запуске:
   - Загрузится SQL.js (~2 МБ, только первый раз)
   - Создастся БД из schema.sql
   - Загрузится план тренировок из plan.csv
4. После загрузки появится интерфейс

### Шаг 4: Использование

1. Выберите **неделю** (1-4) и **день** тренировки
2. Выберите **дату**
3. Введите **ТМ** (тренировочный максимум) для упражнений
4. Введите **вес** и **повторы** для каждого сета
5. Нажмите **💾 Сохранить в БД**

## 🔧 РЕШЕНИЕ ПРОБЛЕМ

### Проблема: SQL.js не загружается

**Решение:**
- Проверьте интернет соединение (нужен только для первого запуска)
- Обновите страницу
- Проверьте консоль браузера (F12) на ошибки

### Проблема: БД не создается

**Решение:**
- Проверьте консоль браузера на ошибки
- Убедитесь, что schema.sql доступен
- Проверьте, что IndexedDB поддерживается браузером

### Проблема: План не загружается

**Решение:**
- Проверьте, что plan.csv доступен
- Проверьте консоль на ошибки
- Попробуйте обновить страницу

### Проблема: Данные не сохраняются

**Решение:**
- Проверьте консоль на ошибки
- Убедитесь, что заполнены вес и повторы
- Проверьте, что IndexedDB работает

### Проблема: PWA не устанавливается на iPhone

**Решение:**
- Используйте Safari (не Chrome или другие браузеры)
- Убедитесь, что сайт открыт через HTTPS
- Проверьте, что manifest.json доступен
