# MESO • Тренировки

Полностью локальное приложение для отслеживания тренировок с использованием SQL.js и карточной раскладкой для iPhone.

## Особенности

- ✅ **Полностью локальное** - все данные хранятся в браузере (IndexedDB)
- ✅ **Работает офлайн** - не требует интернета после первой загрузки
- ✅ **Карточная раскладка** - удобный интерфейс для iPhone
- ✅ **SQL.js** - полноценная SQL база данных в браузере
- ✅ **Экспорт/импорт** - резервные копии в JSON формате
- ✅ **Автоматические расчеты** - RIR, RPE, e1RM вычисляются автоматически
- ✅ **Подсказки** - показывает последние данные по упражнениям

## Быстрый старт

1. Откройте `index.html` в браузере
2. При первом запуске автоматически создастся БД и загрузится план тренировок
3. Выберите неделю и день тренировки
4. Введите вес и повторы - остальное считается автоматически
5. Сохраните данные в БД

## Использование

### Тренировка

1. Выберите **неделю** (1-4) и **день** тренировки
2. Выберите **дату**
3. Введите **ТМ** (тренировочный максимум) для каждого упражнения
4. Введите **вес** и **повторы** для каждого сета
5. Нажмите **Сохранить в БД**

### Подсказки

- **Заполнить** - автоматически подставляет последние данные
- **Рассчитать** - рассчитывает рекомендуемый вес на основе истории

### Экспорт/импорт

- **Экспорт** - сохраняет всю БД в JSON файл для резервной копии
- **Импорт** - восстанавливает БД из JSON файла

## Формат данных

### План тренировок (plan.csv)

CSV формат: `day,exercise,setrep,type,rir_w1,rir_w2,rir_w3,rir_w4,note`

### Экспорт БД

JSON формат с base64-кодированной БД:
```json
{
  "version": 1,
  "data": "base64-encoded-database",
  "timestamp": "2024-11-04T..."
}
```

## Технические детали

### Технологии

- **SQL.js** - SQLite в браузере (WebAssembly)
- **IndexedDB** - хранение БД
- **PWA** - Progressive Web App для установки на iPhone

### Хранение данных

- Все данные хранятся в IndexedDB
- БД автоматически сохраняется при каждом изменении
- Данные сохраняются между сессиями

### Резервные копии

- Экспорт БД в JSON файл
- Импорт БД из JSON файла
- Рекомендуется делать резервные копии регулярно

## Установка на iPhone

1. Откройте сайт в **Safari**
2. Нажмите **Поделиться** → **На экран «Домой»**
3. Приложение установится как PWA

## Структура проекта

- `index.html` - основной файл
- `app.js` - логика приложения
- `db.js` - работа с SQL.js и IndexedDB
- `style.css` - стили для карточной раскладки
- `schema.sql` - схема БД
- `plan.csv` - программа тренировок
- `service-worker.js` - PWA service worker
- `manifest.json` - PWA манифест
